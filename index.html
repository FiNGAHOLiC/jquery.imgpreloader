jquery.imgpreloader.js

jquery.imgpreloader.js is a simple image preloader plugin. $.imgpreloader returns jQuery deferred object.

Example 1 - Load single image

Code

$.imgpreloader({
	ignoreBroken: true,
	paths: [ './img/example1/1.jpg' ]
}).done(function($imgs){
	alert('Loaded!');
	$output.append($imgs);
}).fail(function($imgs){
	alert('Not loaded!');
});

Example 2 - Failed to load single image

Code

$.imgpreloader({
	ignoreBroken: false,
	paths: [ './img/example2/notexist.jpg' ]
}).done(function($imgs){
	alert('Loaded!');
	$output.append($imgs);
}).fail(function($imgs){
	alert('Not loaded!');
	$output.text('Something wrong!');
});
